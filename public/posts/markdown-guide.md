# Markdown Guide: Tips and Tricks

This post will show you how to use various Markdown features to create beautiful blog posts.

## Text Formatting

You can make text **bold**, *italic*, or ***both***. You can also add ~~strikethrough~~ to your text.

## Lists

### Unordered Lists
- First item
- Second item
  - Sub-item 1
  - Sub-item 2
- Third item

### Ordered Lists
1. First step
2. Second step
   1. Sub-step A
   2. Sub-step B
3. Third step

## Code Blocks

You can add inline code like `const greeting = "Hello"` or code blocks:

```javascript
function calculateSum(a, b) {
  return a + b;
}

// Using the function
const result = calculateSum(5, 3);
console.log(result); // Output: 8
```

```python
def greet(name):
    return f"Hello, {name}!"

# Using the function
message = greet("World")
print(message)  # Output: Hello, World!
```
