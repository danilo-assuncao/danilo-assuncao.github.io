# Welcome to My Markdown Blog

This is a sample blog post written in Markdown. This blog system supports all standard Markdown features.

## Features

- **Bold text** and *italic text*
- Lists (like this one!)
- Code blocks
- Images
- And much more...

### Code Example

Here's a simple code example:

```javascript
function greet(name) {
  return `Hello, ${name}!`;
}
```

### Styling

The blog comes with a customizable theme system. You can easily change:

- Colors
- Typography
- Spacing
- Breakpoints

Just modify the `theme.js` file to match your preferred style!

---

Feel free to create your own posts in the `posts` directory using Markdown files.
